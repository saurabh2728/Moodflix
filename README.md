# MoodFlix

MoodFlix is a simple, intuitive web application that provides personalized movie recommendations based on your current mood and preferences. 

## Features
- **Mood-Based Filtering**: Get recommendations tailored to how you are feeling.
- **Customizable Criteria**: Filter movies by energy level, pace, language, category, and who you are watching with.
- **Instant Results**: Fast, on-the-fly movie suggestions based on your selections.
- **Clean & Responsive UI**: A modern, easy-to-use interface that works across devices.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- Vanilla JavaScript

## Getting Started

Since this is a static project, there is no build process or complex setup required.

1. Clone the repository:
   ```bash
   git clone https://github.com/saurabh2728/Moodflix.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Moodflix
   ```
3. Open `index.html` in your favorite web browser.

Enjoy finding your next favorite movie!
